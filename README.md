

# 🌐 Priyadharshini – AI & Data Science Portfolio

A responsive **HTML/CSS/JavaScript portfolio website** showcasing Priyadharshini’s resume, projects, skills, and contact information. Designed with a clean, minimalist UI to highlight academic achievements, technical expertise, and professional experience.


## ✨ Features

- **Home / Hero Section**: Name, tagline, profile picture, and CTA button (“View Projects”).  
- **About Me**: Professional bio, education, and key strengths.  
- **Resume**: Link to PDF resume with download option.  
- **Projects**:
  - AI-Powered Task & Productivity Automation Platform (featured)  
  - Movie Recommendation System  
  - Air Quality Prediction System  
- **Skills**: Showcased with icons and progress indicators.  
- **Contact**: Email form + social links (GitHub, LinkedIn, Email).  
- **Footer**: Minimalist copyright section.  

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript  
- **Styling**: Custom CSS (responsive design)  
- **Deployment**: GitHub Pages / Netlify / Vercel  
- **Version Control**: Git & GitHub  


## 📂 Project Structure

```
portfolio/
├── index.html         # Main portfolio page
├── style.css          # Custom styles
├── script.js          # Optional interactivity
├── assets/            # Images, icons, resume PDF
└── README.md          # Project documentation
```

## ⚡ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Priya162005/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in your browser:
   ```bash
   start index.html   # Windows
   open index.html    # macOS
   ```

## 🌐 Deployment

### GitHub Pages
1. Go to repository settings.  
2. Enable **Pages** → Select branch `main` → `/root`.  
3. Your portfolio will be live at:  
   `https://Priya162005.github.io/portfolio`

### Netlify / Vercel
1. Import repository.  
2. Deploy → Your portfolio is live!
